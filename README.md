# macro-mac-ricky

A native Swift project for building a Mac macro/automation tool.

## Stack
- Language: Swift
- Platform: macOS
- Package Manager: Swift Package Manager

## Goals
- Record keyboard/mouse input
- Replay macros
- Save macro profiles
- Add a simple Mac UI later

## Getting Started

```bash
swift run
```
