# Project Scope

## Purpose
Define the intended scope of the mining bot project before a formal roadmap is created.

## In Scope
The project is expected to explore and eventually support:
- native macOS implementation in Swift
- keyboard and mouse action execution
- configurable action sequences
- visual signal detection
- state-based decision flow
- profile or behavior configuration
- logging and debugging support
- a user-facing interface for controlling execution

## Not Yet Defined
The following areas still require clarification before roadmap planning:
- exact interaction model
- specific detection strategy
- level of configurability in the editor
- final UI shape
- performance targets
- profile format
- deployment/distribution flow

## Guiding Principles
- Keep the architecture modular
- Prefer clarity over premature complexity
- Separate detection, decision, and execution concerns
- Design with future maintainability in mind
- Avoid hardcoding behavior too early

## Outcome of This Phase
By the end of the pre-roadmap phase, the project should have:
- a clearly stated purpose
- a defined problem space
- a high-level architecture direction
- enough detail to build a realistic roadmap
